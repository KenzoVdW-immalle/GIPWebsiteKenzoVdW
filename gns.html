GNS 3

Voor systeembeheer moesten Liam Kathleen en ik een testopstelling maken. We hebben gekozen om deze te maken met GNS 3, een software waarmee je een virtueel netwerk kan maken

Installatie op Linux

  • sudo add-apt-repository ppa:gns3/ppa
  • sudo apt-get update
  • sudo apt-get install gns3-gui

We hebben dit gedaan, maar achteraf gezien hebben we hiervoor niet gekozen. Dit komt omdat we anders er dubbel virtueel te werk zouden gaan. Op de website van GNS3 staat dat het optimaler is om Windows of Mac te gebruiken t.o.v. Linux. Natuurlijk waren we ook al gestart met GNS3 in Windows en dit staat ons meer aan.

Windows

Als je eenmaal GNS3 geïnstalleerd hebt, kan je een image importeren. Bij GNS3 wordt vaak de Cisco IOS image gebruikten wij hebben gekozen voor de Cisco C1700. Je kan makkelijk een link vinden om de image te installeren.

Als je deze image gedownload hebt is het tijd om deze te installeren. Dit kan je makkelijk doen in GNS3 door naar Edit → Preferences → IOS routers → new te gaan, daarna selecteer je de image en wordt die geïmporteerd.

Nu kan je allerlei dingen proberen in het kleine netwerk dat je gemaakt hebt, pingen, console openen,…
